Qué mejoras aporta esta versión

1️⃣ Si un sensor falla el programa no se detiene

2️⃣ Si un sensor falla repetidamente
→ se reinicializa automáticamente

3️⃣ Si varios sensores fallan
→ se reinicia el bus I²C

4️⃣ El GNSS se lee cada 2 segundos
→ evita saturar el bus

5️⃣ Se añade un pequeño retardo entre sensores
→ mayor estabilidad
